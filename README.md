# PreclinicalAD_GutMicrobiome-Inflammation
Scripts used by J. Ryou and M. Gorelik for analysis and visualization of microbiome and fecal ELISA data from healthy and preclinical AD patients.

Shell scripts include version numbers used for each specific tool and were all run on in the High Throughput Computing Facility computing environment. R code runs successfully on R versions 4.4.2 and later.
